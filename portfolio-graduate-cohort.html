Programme Overview

The Graduate Developer Cohort is Corespec's flagship full-stack software engineering programme designed for recent graduates and career changers entering tech. Each cohort combines structured curriculum delivery with real project work — every graduate leaves with a deployed, publicly accessible application in their portfolio.

Curriculum Structure

  • Weeks 1–3: Programming fundamentals (Python and JavaScript), version control (Git), and developer workflow
  • Weeks 4–6: Frontend development — React, responsive design, REST API consumption
  • Weeks 7–9: Backend development — Node.js, Express, PostgreSQL, authentication
  • Weeks 10–12: Full project sprint — teams of 3 build and deploy a complete web application from specification to production

Project Delivery

Each team received a real project brief (sourced from SME partners and NGOs), designed their database schema, built the frontend and backend, and deployed to AWS. Corespec engineers conducted code reviews at each milestone. Participants presented their projects to a panel including external industry professionals.

Career Outcomes

Within 90 days of the cohort ending: all 18 graduates had secured employment or freelance contracts. 12 joined Lagos tech companies (including 3 fintech firms), 4 secured remote contracts with international clients, and 2 founded startups. Average starting salary: ₦180,000/month — 40% above the Lagos tech graduate median.

Graduate Voice

"Before Corespec I had a Computer Science degree but had never actually built anything. Now I have 3 deployed applications on my GitHub, I understand how production systems work, and I have a job I love." — Graduate, Cohort 3
